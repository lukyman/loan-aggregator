Project
 Program that helps accounting department to aggregate monthly loan data into categories of Network, Product and Month.
 The aggregated values are: Total loan per category and number of loans per category.

Programming language
 The program is written in python3. Its easy to run python on all platforms (Linux, Windows and Mac) and with litle configuration to get started.

How to run 
 Open Terminal(Linux or Mac) or Command Prompt (Windows) and write in the below command.
 python3 main.py

Assumptions 
 1. The person using this library is familiar with python programming language

3rd Party Librarier 
 No third party libraries used except built in python features.
